**1. جمع البيانات واستخدامها**
تطبيقنا صُمم ليكون آمناً ويحترم خصوصيتك بالكامل. التطبيق يعمل بشكل مستقل (Offline) ولا يتطلب الاتصال بالإنترنت لأداء وظائفه الخوارزمية. نحن **لا نجمع، ولا نخزن، ولا نشارك** أي بيانات شخصية أو معلومات عن جهازك أو سجلات استخدامك. 

**2. الصلاحيات (Permissions)**
التطبيق لا يطلب أي صلاحيات خاصة للوصول إلى الكاميرا، أو الميكروفون، أو جهات الاتصال، أو الموقع الجغرافي (GPS)، أو ملفاتك الشخصية. 

**3. خدمات الأطراف الثالثة (Third-Party Services)**
التطبيق لا يحتوي على أي حزم برمجية لأطراف ثالثة تهدف إلى تتبع المستخدمين أو عرض الإعلانات المستهدفة. الخوارزميات تعمل محلياً داخل جهازك فقط.
