<p>تطبيقنا صُمم ليكون آمناً ويحترم خصوصيتك بالكامل. التطبيق يعمل بشكل مستقل (Offline) ولا يتطلب الاتصال بالإنترنت لأداء وظائفه الخوارزمية. نحن <strong>لا نجمع، ولا نخزن، ولا نشارك</strong> أي بيانات شخصية أو معلومات عن جهازك أو سجلات استخدامك.</p>

<h3 style="color: #00695c;">2. الصلاحيات (Permissions)</h3>
<p>التطبيق لا يطلب أي صلاحيات خاصة للوصول إلى الكاميرا، أو الميكروفون، أو جهات الاتصال، أو الموقع الجغرافي (GPS)، أو ملفاتك الشخصية.</p>

<h3 style="color: #00695c;">3. خدمات الأطراف الثالثة (Third-Party Services)</h3>
<p>التطبيق لا يحتوي على أي حزم برمجية لأطراف ثالثة تهدف إلى تتبع المستخدمين أو عرض الإعلانات المستهدفة. الخوارزميات تعمل محلياً داخل جهازك فقط.</p>
